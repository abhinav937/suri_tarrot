# Images Folder

Place your images here.

## Recommended Image Types

- **Logo**: `logo.png` or `logo.svg`
- **Hero Background**: `hero-bg.jpg` or `hero-bg.png`
- **Service Images**: 
  - `mehendi-service.jpg`
  - `tarot-reading.jpg`
- **Gallery Images**: Place in subfolders if needed
- **Favicon**: `favicon.ico` (place in root directory)

## Image Optimization Tips

- Use compressed images (JPEG for photos, PNG for graphics with transparency)
- Recommended sizes:
  - Hero images: 1920x1080px
  - Service cards: 800x600px
  - Gallery thumbnails: 400x400px
- Consider using WebP format for better compression
