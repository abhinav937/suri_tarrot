# Fonts Folder

Place custom font files here if you want to use self-hosted fonts.

## Current Setup

The website currently uses Google Fonts:
- **Playfair Display**: For headings
- **Poppins**: For body text

## If You Want to Use Custom Fonts

1. Add your font files (.woff2, .woff, .ttf) to this folder
2. Update the `@font-face` declarations in `assets/css/styles.css`
3. Update the font-family references in the CSS

## Font File Formats

- **WOFF2**: Best compression, modern browsers
- **WOFF**: Good compression, wider browser support
- **TTF/OTF**: Fallback for older browsers
